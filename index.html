Почему создавать API на GOLANG?

В новом курсе https://purpleschool.ru/course/go-advanced мы подробно разберемся с тем, как работает Go под капотом, изучим гарутины и создадим собственное API на Go.


Почему стоит выбрать Go для проектирования API?

  1. Производительность: Go способен обрабатывать большое количество запросов в секунду.
  2. Конкурентность: Благодаря гарутинам, Go позволяет выполнять множество операций параллельно.
  3. Богатая стандартная библиотека: Go предлагает множество встроенных функций для разработки, включая HTTP роутинг.

Основные темы курса:

  1. Понимание Go runtime: изучение структуры бинарников, работы garbage collector и управления памятью.
  2. Работа с гарутинами: изучение интерфейсных каналов и синхронизации гарутин.
  3. Разработка API с использованием базы данных Postgres и ORM GORM.
  4. Работа с промежуточными обработчиками, авторизацией, логированием.
  5. Написание тестов для API.
  6. Освоение контекста и его применения в Go.

Для успешного освоения курса требуется базовое знание Go. Если вы уже прошли вводный курс по Go, вы готовы начать! Этот курс отлично подойдет для тех, кто хочет глубже понять работу гарутин или создать полноценный API на Go.
